contenedores

un contenedor fluido en bootstrap que se crea utilizando la clase conteiner-fluid.Este tipo de contenedor ocupa el 100% de el ancho del viewport, lo que significa que se expande para adaptarse a cualquier tamaño de pantalla .Es ideal para diseños que necesitan aprovechar al maximo el espacio horizontal disponible, sin importar el dispocitivo que se este utilizando
